Describe which are the appropriate filetypes for websites.

There are a number of different filetypes for websites, some of the more common ones include: Html which this is used in website creation and are is also used by end users in order to navigate around a web page. Asp is used to create dynamic and interactive web pages which are not affected by the type of browser a person is using. Php is used to develop either static or dynamic web pages or applications. Css which is a type of file that takes care of how the website looks to the end user.

As well as the web file types, there are also different files types based on the content on your web page, Some of the more common and appropriate types are as follows: For images, the file types of jpeg, png, jpg. For audio file types there are wav, mp3, aac. For video file types there are : avi, mpg, mpeg, mov, wmv. For documents and text, there are the file types txt, doc, docx pdf for some of the more common types.
